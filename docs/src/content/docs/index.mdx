## Quickstart

Please do try and read this guide, but if you're in a hurry and want to get
started quickly:

<Tabs>
  <TabItem label="bash/zsh">
  ```bash
  bash -c "$(curl https://bebop.sh)"
  ```
  </TabItem>

  <TabItem label="powershell">
  ```powershell
  irm https://bebop.sh | iex
  ```
  </TabItem>
